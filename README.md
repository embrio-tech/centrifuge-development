A development repository to combine and manage all necessary services to easily develop the centrifuge insights dashdoard.

## Getting Started

### Prerequisites

- docker
- docker-compose

### Initialize

    $ cp .env.sample .env

### Start

    $ docker-compose up

### Scrap

    $ docker-compose down -v
